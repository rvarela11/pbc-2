# Patriot Boot Camp Alumni Directory

# About

* A public-facing online directory of alumni and their companies.

* A CRM back-end to power it.

* Technology Stack:

  * Node.js/Express
  * PostgreSQL
  * AWS (Elastic Beanstalk for Server, S3 for Front-End)
